# Noteworthy

The power of LLMs brought to an interface.

## Features

1. File upload
   -  Copy file content
   -  Summarise file content
   -  Consolidate content from multiple files
2. Text highlighting
   -  Re-write
   -  Summarise
   -  Create examples
3. Multimedia
   -  Summarise audio (from links)
   -  Create images/diagrams of content

## Run

To run the dev site:

```bash
npm install
npm run dev
```
